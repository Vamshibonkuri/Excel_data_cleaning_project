**Excel Data Cleaning Project**

## 🔹 Project Overview

This project focuses on cleaning and transforming raw Excel data into a structured and analysis-ready format. The goal is to ensure data accuracy, consistency, and usability for further analysis.


## 🔹 Tools Used

* Microsoft Excel


## 🔹 Dataset

* Raw dataset: `messy_data.xlsx`
* Cleaned dataset: `cleaned_data.xlsx`


## 🔹 Data Cleaning Steps

* Removed duplicate records
* Handled missing/null values
* Standardized text (e.g., uppercase/lowercase)
* Converted data types (dates, percentages, numbers)
* Fixed inconsistent formatting
* Trimmed extra spaces


## 🔹 Key Features

* Improved data quality
* Ready for analysis and visualization
* Structured dataset for business insights


## 🔹 Before vs After

Raw data contained inconsistencies, missing values, and formatting issues. After cleaning, the dataset became structured, consistent, and analysis-ready.


## 🔹 Use Case

This cleaned dataset can be used for:

* Data analysis
* Dashboard creation (Power BI / Excel)
* Business decision-making
